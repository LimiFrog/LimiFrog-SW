
This directory contains information on using the provided software package within the "System Workbench for STM32" IDE.

It is an Eclipse-based IDE (Integrated Development Environment) which relies on GNU tools for compiling/linking and uses OpenOCD to flash and debug the board. It runs under Windows or Linux.

Users of other Eclipse-based IDE, as well as users of OpenOCD whatever the development environment, may also find some relevant information here.
