FatFS Formatting
================

Initialize board

Upon pressing one of the 2 push-button switches,
format Data Flash as FAT File System using default settings
and write a short log file on the formatted Flash Device
****  CAUTION:  Prior Contents will be lost *****

Then Configure board as Mass Storage Device (8MB)
USB Device Stack runs on STM32, Physical Storage space is Data Flash.
File FS_Formt.log should be visible from PC by USB.
